## 📌 Project Overview

**LightHouse** is a full-stack travel and tourism website built during **Smart India Hackathon 2023** by **Arnav Shukla** and teammate **Priyanshu**. The platform serves as a comprehensive travel guide for India — covering states, monuments, beaches, temples, and natural wonders — with the goal of making great travel experiences accessible to everyone.

No amazing traveling spot left unexplored. That's the LightHouse promise.

---

## ✨ Key Features

- 🗺️ **State-wise Travel Guides** — Explore destinations organized by Indian state
- 🏛️ **Landmark Coverage** — Temples, beaches, monuments, forests, and more
- 📷 **Rich Visual Gallery** — High-quality images for each destination
- 🎬 **Video Playback** — Immersive travel video experience
- 📱 **Responsive Design** — Optimized for desktop and mobile browsers
- 🔍 **Easy Navigation** — Clean, intuitive UI for discovering new places

---

## 📍 Destinations Covered

The platform features destinations across India, including:

- **Goa** — Anjuna Beach, Baga Beach
- **Gujarat** — Dwarkadhish Temple, Gir Forest, Somnath Temple, Sabarmati Ashram, Junagarh
- **Rajasthan** — Hawamahal, Red Fort, Golden Temple
- **Delhi** — Iconic landmarks and heritage sites
- **Himachal Pradesh** — Hill stations and scenic landscapes
- **Maharashtra** — Dudhsagar Falls, Khandala
- **And many more...**

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Arnavs10/SIH_2023_LightHouse_Website.git
   ```
2. Open `TRAVEL AND TOURISM.html` in any modern browser
3. No build step required — it's a pure HTML/CSS/JS project

## 🏆 About Smart India Hackathon 2023

Smart India Hackathon (SIH) is India's biggest open innovation model — a national-level hackathon where students build solutions to real-world problems. LightHouse was built as part of this event, focusing on promoting tourism and making travel knowledge widely accessible.
